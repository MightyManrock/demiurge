# Canonical Occupation List by Stratum

Each occupation has a two-column alias table: civilization scale on the left, display name on the right. "N/A" rows indicate the occupation does not meaningfully exist at that scale. A single "—" dash indicates that this should be blank and default to the occupation name.

Two additional alias dimensions are not yet fully specified: **government type** (autocracy, oligarchy, theocracy, democracy, corporate, military, clan-based) and **location type** (terrestrial, spacer). These are TODO for full completion.
